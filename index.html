About Us

Taskly is a web-based activity and progress tracker designed to help users organize and manage tasks efficiently. The system allows admins to create activities with stages and deadlines, while users can submit their work and track completion progress.

With its clean interface and organized dashboard system, Taskly provides a convenient solution for monitoring tasks, submissions, and overall progress.

Features

Activity Management

Create activities with multiple stages, instructions, and deadlines.

Deadline Tracking

Stay updated with clear due dates and automatic overdue monitoring.

Admin Monitoring

Monitor all client accounts, submissions, and overall completion rates.

Stage Submission

Users can open each stage, upload work, and mark progress as completed.

Progress Overview

See progress percentages and stage completion in one organized view.

Local Saving

Data is stored in the current browser using local storage for quick testing.
